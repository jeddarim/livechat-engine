# 💬 Realtime Chat App

A full-stack realtime chat application that enables users to communicate instantly through channels using WebSockets and REST APIs.

---

## 🚀 Features

* 💬 Realtime messaging with WebSockets
* 📢 Create and join chat channels
* 🔄 Instant updates across users
* 🌐 REST API integration
* ⚡ Smooth and responsive UI

---

## 🛠️ Tech Stack

### Frontend

* React
* TypeScript
* Material UI
* Socket.io Client

### Backend

* Node.js
* Express.js
* REST API
* Socket.io

---

## 🧠 What I Learned

* Implementing realtime communication using WebSockets
* Managing bidirectional data flow with Socket.io
* Building RESTful APIs with Node.js
* Structuring full-stack applications
* Handling client-server synchronization

## 📂 Project Structure

* `/client` → React frontend
* `/server` → Node.js backend

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/realtime-chat-app.git
cd realtime-chat-app
```

### 2. Setup Backend

```bash
cd server
npm install
npm run start
```

### 3. Setup Frontend

```bash
cd client
npm install
npm start
```
