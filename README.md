# Realtime Chat App

Full-stack realtime chat application.

## Tech Stack
Frontend:
- React
- TypeScript
- Material UI
- Socket.io

Backend:
- Node.js
- REST API
- WebSockets

## Features
- Create channels
- Realtime messaging
- Socket communication
- REST endpoints

## Run
Backend:
npm install
npm run start

Frontend:
npm install
npm start
